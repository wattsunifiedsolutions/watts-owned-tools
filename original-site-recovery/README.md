# Original Watts Unified site recovery

This directory is generated from the July 10, 2026 live backup of the original GHL-published Watts Unified site.

Run `scripts/build-original-site-recovery.ps1` from the repository root to rebuild `public/` with the original 23 routed pages and locally captured assets. The generated output deploys only to the existing Cloudflare Pages project `watts-unified-rebuild` attached to `wattsunified.com` and `www.wattsunified.com`.
